Vector

A vector is a sequence of values of the same type

(e.g., numeric, character, date, …).

You use c() function to create a vector.

Numeric

a <- c(0, 1, 2)

Character

b <- c("A", "B", "C")

Object Classes

a
[1] 0 1 2
class(a)
[1] "numeric"
b
[1] "A" "B" "C"
class(b)
[1] "character"

Why the values of the same type?

Some operations work on only specific types.

a * a
[1] 0 1 4
b * b

Data Frame

You can create a data frame using data.frame() function.

df <- data.frame(a, b)
df
  a b
1 0 A
2 1 B
3 2 C

Or you can type individual elements.

data.frame(a = c(0, 1, 2), b = c("A", "B", "C"))
  a b
1 0 A
2 1 B
3 2 C

Tibble

A tibble is an enhanced version of data frame used in Tidyverse.

tb <- tibble(a, b)
tb
# A tibble: 3 × 2
      a b    
  <dbl> <chr>
1     0 A    
2     1 B    
3     2 C

Common operations with a tibble or a data frame

You can pull out a column as a vector, using $.

tb$a
[1] 0 1 2

Calculate the mean value of a vector

mean(tb$a)
[1] 1

Show column names as a vector

names(tb)
[1] "a" "b"

Show the number of rows

nrow(tb)
[1] 3

Show the first two rows

head(tb, 2)
# A tibble: 2 × 2
      a b    
  <dbl> <chr>
1     0 A    
2     1 B

Without the second argument, it shows 6 rows.

head(sale)
